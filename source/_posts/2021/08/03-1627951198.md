## 题目

<https://leetcode.com/problems/reverse-integer/>

### 大意

给定 32 位整数 `x`，返回反转后的数字，如果存在溢出，则返回 `0`。

<!-- more -->

## 解法一：模运算

通过模运算来反转 `x` 即可。

需要注意的是，判断反转后的数字 `res` 是否会溢出，这个判断不能在每次模运算之后（因为如果溢出，此时已经溢出），而应该在每次模运算之前，判断 `res` 是否超出了 `INT_MAX/10`。

### 复杂度分析

- 空间：`O(1)`

- 时间：`O(n)`

### C++ 实现

```cpp
class Solution {
public:
    int reverse(int x) {
        int res = 0;
        while (x != 0) {
            if (abs(res) > INT_MAX/10) {
                return 0;
            }
            res = res * 10 + x % 10;
            x /= 10;
        }
        return res;
    }
};
```
